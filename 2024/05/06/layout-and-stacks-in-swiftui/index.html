Layout and Stacks in SwiftUI

06 May 2024

In SwiftUI, layout is managed through the use of stacks—vertical (VStack) and horizontal (HStack)—which allow for flexible arrangement of views. Stacks simplify the process of organizing UI elements, making it easy to create complex layouts. Let’s explore how to utilize stacks effectively in SwiftUI.

VStack and HStack

VStack

The VStack stacks views vertically, arranging them in a column. This is useful for displaying content in a top-to-bottom order.

import SwiftUI

struct ContentView: View {
    var body: some View {
        VStack {
            Color
                .purple
                .frame(width: 100, height: 100)
            Color
                .pink
                .frame(width: 100, height: 100)
        }
    }
}

HStack

The HStack stacks views horizontally, placing them in a row. This is ideal for arranging elements from left to right.

import SwiftUI

struct ContentView: View {
    var body: some View {
        HStack {
            Color
                .purple
                .frame(width: 100, height: 100)
            Color
                .pink
                .frame(width: 100, height: 100)
        }
    }
}

Read more about HStack

ZStack

The ZStack stacks views on top of each other in a Z-axis arrangement, allowing for overlapping content.

import SwiftUI

struct ContentView: View {
    var body: some View {
        ZStack {
            Rectangle()
                .fill(Color.purple)
                .frame(width: 200, height: 200)
            Circle()
                .fill(Color.pink)
                .frame(width: 150, height: 150)
        }
    }
}

Using Spacer

The Spacer view expands to push views to the edges of a stack, providing flexible spacing between elements.

import SwiftUI

struct ContentView: View {
    var body: some View {
        VStack {
            Color
                .purple
                .frame(width: 100, height: 100)
            Spacer()
            Color
                .pink
                .frame(width: 100, height: 100)
        }
    }
}

Nesting Stacks

You can nest stacks to create more complex layouts. Here’s an example of combining VStack and HStack.

import SwiftUI

struct ContentView: View {
var body: some View {
        VStack {
            HStack {
                Text("1")
                    .font(.largeTitle)
                    .foregroundColor(.white)
                    .frame(width: 100, height: 100)
                    .background(Color.purple)

                    
                Spacer()
                Text("2")
                    .font(.largeTitle)
                    .foregroundColor(.white)
                    .frame(width: 100, height: 100)
                    .background(Color.pink)
            }
            Spacer()
            HStack {
                Text("3")
                    .font(.largeTitle)
                    .foregroundColor(.white)
                    .frame(width: 100, height: 100)
                    .background(Color.orange)

                    
                Spacer()
                Text("4")
                    .font(.largeTitle)
                    .foregroundColor(.white)
                    .frame(width: 100, height: 100)
                    .background(Color.green)
            }
        }.padding()
    }
}
