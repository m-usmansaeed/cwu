Mastering Lists in SwiftUI

10 May 2024

Lists are a cornerstone for displaying structured data in SwiftUI applications. They provide a flexible and efficient way to present collections of items, from grocery lists to intricate navigation menus. This article delves into the core concepts of Lists in SwiftUI, guiding you through essential code examples and practical applications.

Creating Basic Lists

The most straightforward way to create a List is by providing individual views for each row. Hereâ€™s an example:

struct FruitList: View {
    let fruits = ["Apple", "Banana", "Cherry"]

    var body: some View {
        List {
            ForEach(fruits, id: \.self) { fruit in
                Text(fruit)
            }
        }
    }
}

Explanation:

  1. We define a FruitList struct that holds an array of fruit names.
  2. The List container displays the content within.
  3. ForEach iterates over the fruits array, assigning each element a unique identifier (\.self).
  4. Inside the loop, a Text view displays the current fruit name for each row.

Dynamic Lists with Identifiable Data

For larger datasets, itâ€™s more efficient to create Lists on demand using identifiable data. This example demonstrates how:

struct SongList: View {
    let songs = [
        Song(title: "Bohemian Rhapsody", artist: "Queen"),
        Song(title: "Imagine", artist: "John Lennon"),
        Song(title: "Hallelujah", artist: "Leonard Cohen")
    ]

    var body: some View {
        List {
            ForEach(songs) { song in
                HStack {
                    Text(song.title)
                    Spacer()
                    Text(song.artist)
                        .foregroundStyle(.secondary)
                }
            }
        }
    }
}

struct Song: Identifiable {
    let id = UUID() // Unique identifier
    let title: String
    let artist: String
}

Explanation:

  1. We define a Song struct with properties for title, artist, and a unique identifier (id).
  2. An array of Song instances is created.
  3. The List displays content dynamically.
  4. ForEach iterates over the songs array, where each song becomes available within the loop.
  5. An HStack arranges Text views for title and artist within a row.

Read more about Identifiable and UUID

Customizing List Appearance

SwiftUI offers various ways to customize the visual style of Lists. Hereâ€™s an example using the listStyle modifier:

struct GroceryList: View {
    let groceries = ["Milk", "Bread", "Eggs", "Cheese"]

    var body: some View {
        List {
            ForEach(groceries, id: \.self) { item in
                Text(item)
            }
        }
        .listStyle(PlainListStyle()) // Set list style to plain
    }
}

Explanation:

  1. The listStyle modifier applies a specific visual style to the List.
  2. In this case, PlainListStyle() creates a simple, unstyled List.

Adding Navigation with Lists

Lists can seamlessly integrate with navigation to create hierarchical app structures. Hereâ€™s an example:

struct LandmarkList: View {
    let landmarks = [
        Landmark(
            name: "Hagia Sophia",
            imageName: "hagiaSophia"),
        Landmark(
            name: "Great Wall of China",
            imageName: "greatwall")
    ]
    
    var body: some View {
        NavigationStack {
            List {
                ForEach(landmarks) { landmark in
                    NavigationLink(
                        destination: LandmarkDetail(
                            landmark: landmark)
                    ) {
                        LandmarkItem(
                            landmark: landmark)
                    }
                }
            }
        }
        .navigationTitle("Landmarks")
        
    }
}

struct Landmark: Identifiable {
    let id = UUID()
    let name: String
    let imageName: String
}

To read more about Navigation check out Navigation in SwiftUI

Lets create a simple LandmarkItem component.

struct LandmarkItem: View {
    let landmark: Landmark
    
    var body: some View {
        HStack {
            Image(landmark.imageName)
                .resizable()
                .frame(width: 100, height: 100)
                .clipShape(Circle())
                .aspectRatio(contentMode: .fill)
            Text(landmark.name)
                .font(.title3)
        }
    }
}

And finally lets create a simple LandmarkDetail component.

struct LandmarkDetail: View {
    let landmark: Landmark
    
    var body: some View {
        VStack {
            Image(landmark.imageName)
                .resizable()
                .aspectRatio(contentMode: .fit)
            Text(landmark.name)
                .font(.title)
        }
    }
}
